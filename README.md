# UIQA

UIQA is a UI delivery QA skill for SaaS, ERP, CRM, finance-tax, B-side dashboards, admin systems, and design-to-code handoffs.

It reviews:

- Responsive behavior
- Auto Layout / CSS layout quality
- Spacing, typography, shadow, border, and surface quality
- Component consistency and state coverage
- Business UX and task completion
- Delivery score, UX risk, P0/P1/P2 severity
- Jira / Feishu-ready fix tasks
